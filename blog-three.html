Understanding Django Framework

Posted by James Bumanlag


What is Django?

Django is a high-level, full-stack web framework for building robust and scalable web applications. Written in Python, it emphasizes the DRY (Don't Repeat Yourself) principle and convention over configuration, making it a great choice for rapid development.

Key Features of Django

  1. Batteries-Included Framework
    • Comes with built-in-tools like an ORM, admin interface, from handling, and authentication.
    • Reduces the need for third-party dependencies.

  2. MVC(Model-View-Controller)
    • Django follows MTV (Model-Template-View) pattern
      • Model: Manages the database schema and interactions.
      • Template: Handles presentation with HTML templates.
      • View: Contains business logic, processing request, and returning responses

  3. Object-Relational Mapping (ORM)
    • Simplifies database interactions using Python objects
    • Compatible with various database like SQLite, PostgreSQL, MySql and Oracle.

  4. Admin Interface
    • Automatically generates an admin panel for managing models.
    • Highly customizable for specific requirements

  5. Authentication System
    • Built-in tools for user management, login/logout, password management and session handling
